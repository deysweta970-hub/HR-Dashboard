# HR-Dashboard
Developed an interactive HR Analytics Dashboard in Tableau to analyze employee headcount, attrition, salary distribution, age groups, department performance, and leave balance. Improved HR decision-making through clear KPIs, trend charts, and departmental insights.

# Project Objective
To analyze overall HR performance by visualizing employee demographics, departmental salary distribution, attrition trends, performance ratings, and leave balance, enabling management to understand workforce health and optimize HR strategies.
