# HR-Dashboard
Developed an interactive HR Analytics Dashboard in Tableau to analyze employee headcount, attrition, salary distribution, age groups, department performance, and leave balance. Improved HR decision-making through clear KPIs, trend charts, and departmental insights.

<a href="https://github.com/deysweta970-hub/HR-Dashboard/commit/77f81d0ea42b7ad60e06690ba0a13b4840d35e31">Hr dashboard</a>
# Objective
To visualize employee demographics and workforce distribution.
To identify resignation trends and attrition patterns.
To track department-wise performance and salary allocation.
To provide actionable HR insights through interactive Tableau dashboards.
